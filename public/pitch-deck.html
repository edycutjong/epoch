Slide 07 // Integration Depth

5 Terminal 3 Host API Methods

1
time/clock

Queries CPU monotonic clock for tamper-proof time-gating. The heartbeat that can't lie.

2
otp

Issues and verifies liveness OTP codes. Proof the user is still alive and in control.

3
stash

Seals vault keys and digital records inside Intel TDX secure storage. Zero plaintext exposure.

4
contracts-call

Chains multiple executor contracts with full rollback guarantees. All-or-nothing succession.

5
http-with-placeholders

Contacts beneficiaries blindly. PII substituted at the enclave edge — never logged, never exposed.
